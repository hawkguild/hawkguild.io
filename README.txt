WebApp POS ร้านอาหารใต้แบบเต็มระบบ

วิธีใช้งาน
1. ดับเบิลคลิก run-pos.bat
2. โปรแกรมจะเปิด index.html ในเบราว์เซอร์
3. ข้อมูลถูกเก็บในเครื่องผ่าน localStorage ของเบราว์เซอร์

หน้าจอที่มีในระบบ
- Dashboard: สรุปยอดขายวันนี้ ยอดออนไลน์ มูลค่า Stock และวัตถุดิบใกล้หมด
- ขายหน้าร้าน: เลือกโต๊ะ ผู้ขาย เมนู ชำระเงิน และพิมพ์ใบเสร็จ
- ออเดอร์ออนไลน์: รับออเดอร์ ShopeeFood และ LINE MAN ส่งเข้าหน้าขายทันที
- บิลและรายงาน: ยอดขายวันนี้ ประวัติบิล และพิมพ์ใบเสร็จซ้ำ
- จัดการร้านค้า: ชื่อร้าน สาขา ที่อยู่ เบอร์โทร ภาษี และค่าบริการ
- หัวบิล: ชื่อหัวบิล เลขเครื่อง POS ข้อความหัวบิลและท้ายบิล
- Stock: จำนวนคงเหลือ หน่วย ต้นทุนต่อหน่วย มูลค่าคงเหลือ และจุดสั่งซื้อ
- ผู้ขายเข้ากะ: เพิ่มผู้ขาย เปิด/ปิดกะ และเลือกผู้ขายประจำกะ
- จัดการเมนู: เพิ่ม/แก้ไขเมนู ราคา หมวด สถานะ และอัปโหลดรูปอาหาร
- Icon โปรแกรม: assets/app-icon.svg และ assets/app-icon-512.png ใช้เป็น favicon และ app icon

ข้อแนะนำ
- ใช้ปุ่มสำรองข้อมูลเพื่อเก็บไฟล์ JSON เป็นระยะ
- รูปอาหารจะถูกเก็บในเบราว์เซอร์ ควรใช้รูปขนาดไม่ใหญ่มากเพื่อให้โปรแกรมเปิดเร็ว
